# Gimme Golf — Dark Mode Style Guide

## Design Direction

Dark mode should feel like playing golf late in the evening or sitting in a clubhouse after the round.

The UI should feel:

- calm
- premium
- minimal
- focused
- easy on the eyes
- readable outdoors at night

The design inspiration is similar to Spotify’s dark mode:

- deep background surfaces
- bright but restrained accent colors
- minimal color noise
- strong typography

The interface should feel like **a modern digital scoreboard under soft lighting**.

---

# Core Dark Mode Principles

## 1. True Dark Backgrounds

Avoid mid-gray backgrounds across the entire UI.

Instead use layered surfaces:

- deep black background
- slightly lighter panels
- clear card elevation

This creates visual hierarchy while keeping the UI elegant.

---

## 2. Emerald Accent

Your emerald green remains the primary accent.

It should appear in:

- selected states
- buttons
- score selections
- highlights
- navigation emphasis

Dark mode should still feel like **Gimme Golf**, not a generic dark theme.

---

## 3. Minimal Color Use

The interface should rely mostly on:

- black
- dark gray
- emerald
- white text

Accent colors should only appear for:

- awards
- warnings
- chaos effects

---

- page background
- navigation background

- section panels

Main card background.

- player cards
- challenge cards
- recap panels
- preset cards

# Component Styling

## Cards

Card appearance should be subtle but defined.

background: `#1C1C1C`  
border: `1px solid #2A2A2A`  
radius: 16px  
shadow: extremely subtle

Avoid heavy elevation.

---

## Buttons

### Primary Button

background: emerald  
text: white

Example:

Start Round  
Next Hole

---

### Secondary Button

background: transparent  
border: `1px solid #2A2A2A`  
text: white

---

## Score Buttons

Score buttons should be:

- large
- easy to tap
- highly visible when selected

Default state:

background: dark surface

Selected state:

background: emerald  
text: black

---

# Motion & Feedback

Dark mode should emphasize subtle glow-style feedback.

Examples:

- emerald highlight pulse when selecting a card
- gentle glow on score selection
- soft highlight when a challenge completes

Avoid flashy animation.

---

# Special Moments

Dark mode gives you a great opportunity for dramatic moments.

Examples:

## Awards
Gold highlight + Pirata One heading.

## Featured Hole
Emerald banner glow.

## Chaos Event
Red accent flash.

These should feel like **small dramatic beats**, not visual chaos.

---

# Accessibility

Dark mode must maintain strong contrast.

Guidelines:

- white text on dark surfaces
- avoid gray text on black backgrounds
- large tap targets
- clear selected states

Users should be able to read the UI easily at night or indoors.

---

# Design Summary

Dark mode should feel like:

- a clubhouse scoreboard at night
- calm, elegant, and modern
- minimal color noise
- emerald accents guiding attention

The visual identity should feel:

“golf at night, but still premium.”

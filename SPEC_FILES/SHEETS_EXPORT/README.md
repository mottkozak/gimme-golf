# Google Sheets Export

- `cards_all_packs.csv`: all cards with `packId` + `packName` columns.
- `cards_classic.csv`, `cards_chaos.csv`, `cards_props.csv`, `cards_curse.csv`, `cards_style.csv`, `cards_novelty.csv`, `cards_hybrid.csv`: one file per pack.

Import to Google Sheets:
1. Open a blank Google Sheet.
2. File -> Import -> Upload.
3. Choose one of the CSV files.
4. For all packs in one sheet, use `cards_all_packs.csv`.
